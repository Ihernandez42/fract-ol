Definiciones de funciones básicas de la mlx 

mlx_init: Inicializa la conexión con el servidor gráfico.

mlx_new_window: Crea una nueva ventana en la pantalla.

mlx_pixel_put: Coloca un píxel en una ubicación específica de la ventana.

mlx_loop: Inicia el bucle de eventos para manejar interacciones con la ventana.

mlx_mouse_hook: Establece una función de retrollamada para manejar eventos del mouse.

mlx_key_hook: Establece una función de retrollamada para manejar eventos de teclado.

mlx_expose_hook: Establece una función de retrollamada para manejar eventos de exposición (por ejemplo, cuando la ventana se hace visible).

mlx_loop_hook: Establece una función de retrollamada que se ejecuta en cada iteración del bucle principal.

mlx_int_anti_resize_win, mlx_int_do_nothing, mlx_int_wait_first_expose, mlx_int_get_visual, mlx_flush_event, mlx_string_put, mlx_set_font: Funciones internas utilizadas para manejar diferentes aspectos de la interfaz gráfica.

mlx_new_image: Crea una nueva imagen en memoria.

mlx_get_data_addr: Obtiene la dirección de memoria del primer byte de la imagen.

mlx_put_image_to_window: Dibuja una imagen en una ventana.

mlx_get_color_value: Obtiene el valor de color correspondiente a un color RGB.

mlx_clear_window: Borra el contenido de una ventana.

mlx_xpm: Funciones relacionadas con el manejo de archivos de mapa de píxeles XPM.

mlx_int_str_to_wordtab: Función interna para dividir una cadena en palabras.

mlx_destroy_window: Destruye una ventana creada previamente.

mlx_int_param_event, mlx_int_set_win_event_mask, mlx_hook: Funciones internas para manejar eventos y ganchos.

mlx_rgb: Convierte valores RGB en un valor de color de 32 bits.

mlx_destroy_image: Destruye una imagen creada previamente.

mlx_mouse: Funciones relacionadas con el manejo del mouse.

mlx_screen_size: Obtiene el tamaño de la pantalla.

mlx_destroy_display: Cierra la conexión con el servidor gráfico.
